# Java_prac
자바 공부를 위한 연습
## 소개
본격적인 자바 공부를 하기에 앞서 각 클래스, 객체와의 관계를 잘 이해하기 위해 연습한 **연습노트**입니다.

## 🚗[자동차](https://github.com/Hiluxy/Java_prac/tree/master/car_ex/src)
* 자동차로 클래스, 메서드, 객체 관계를 이해합니다.  
* car클래스: 속성(색, 차종, 스피드), 동작(go() 등)  
* cargame 클래스: 메인메서드  
(메인메서드는 제일 먼저 실행됩니다)
